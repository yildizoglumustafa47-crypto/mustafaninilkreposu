# Proje Fikri

## Proje Adı
To-Do List Uygulaması

## Amaç
Kullanıcının günlük yapılacak işlerini ekleyebileceği basit bir web uygulaması geliştirmek.

## Kullanılan Teknolojiler
- HTML
- CSS
- JavaScript

## Özellikler
- Görev ekleme
- Görev silme
- Görevleri listeleme
- Tamamlandı olarak işaretleme

## Neden Bu Proje?
Bu proje, web geliştirmeye yeni başlayanlar için temel kavramları öğrenmek adına uygundur.
